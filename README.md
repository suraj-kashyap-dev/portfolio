# Suraj Kashyap — Portfolio

Personal portfolio website for **Suraj Kashyap**, AI-First Full-Stack Engineer.

## Stack

Built with plain HTML, CSS, and vanilla JavaScript — no frameworks, no build tools, no dependencies.

- Google Fonts (Syne, Fira Code, DM Sans) loaded via CDN
- All styles inline in `<style>` tag
- All scripts inline at bottom of `<body>`

## Structure

Single file: `index.html`

The page is divided into five sections:

| # | Section | Description |
|---|---------|-------------|
| 01 | About | Bio and core expertise cards |
| 02 | Experience | Work history timeline (Webkul, Opmac) |
| 03 | Skills | Tech stack with animated progress bars |
| 04 | Contact | Links to GitHub, LinkedIn, email, phone |

## Features

- Custom animated cursor (desktop only)
- Hamburger menu for mobile navigation
- Scroll-triggered reveal animations
- Animated skill progress bars on scroll
- Ambient blob background with CSS animations
- Terminal-style typing animation in hero card
- Fully mobile responsive (breakpoints at 900px, 768px, 480px)

## Usage

Just open `index.html` in any browser — no server or build step needed.

```bash
open index.html
# or
python -m http.server 8000
```

## Customization

All CSS variables are defined at the top of the `<style>` block:

```css
:root {
  --cyan: #00e5ff;
  --violet: #a855f7;
  --green: #10b981;
  /* ...etc */
}
```
